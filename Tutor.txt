GitHub
Добавление файлов в:
git add .

Коммит изменений (вы уже сделали это, но для завершенности процесса):
git commit -m "комментарий"

Отправка изменений в удаленный репозиторий на GitHub:
git push origin <branch_name> ИЛИ git push
Замените <branch_name> на название вашей ветки, например, main или master. Если вы работаете с основной веткой и хотите отправить изменения туда, то используйте:

git push origin main
